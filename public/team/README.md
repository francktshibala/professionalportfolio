# Team Photos Folder

Drop your team member photos here.

## Photo Guidelines:

### Format:
- **File type:** JPG or PNG
- **Recommended size:** 400x400px (square)
- **Aspect ratio:** 1:1 (will be displayed as circles)
- **File size:** Keep under 500KB for fast loading

### Naming Convention:
Please name files clearly, for example:
- `franck-tshibala.jpg` (Founder/CEO)
- `john-doe.jpg` (CTO)
- `jane-smith.jpg` (Advisor)

### What to Provide:
Once you've added photos, provide:
1. **Name:** Full name
2. **Title:** Role/Position
3. **Bio:** 1-2 lines of credentials/background
4. **Photo filename:** Exact filename you used

## Example:
```
Name: Franck Tshibala
Title: Founder & CEO
Bio: BYU-Idaho CS Student, Church Employee, Building solutions for 1.5B ESL learners globally
Photo: franck-tshibala.jpg
```
