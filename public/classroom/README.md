# Classroom Photos Folder

Drop your classroom/student photos here showing BookBridge in action at INX Academy.

## Photo Guidelines:

### Format:
- **File type:** JPG or PNG
- **Recommended size:** 800x600px or larger (landscape orientation preferred)
- **Aspect ratio:** 4:3 or 16:9 works well
- **File size:** Keep under 1MB for fast loading

### Naming Convention:
Please name files descriptively, for example:
- `inx-academy-1.jpg`
- `inx-academy-2.jpg`
- `inx-academy-3.jpg`
- `inx-academy-4.jpg`
- `inx-academy-5.jpg`

Or use more descriptive names like:
- `inx-students-reading.jpg`
- `inx-classroom-tablets.jpg`
- `inx-group-learning.jpg`

### What to Provide After Upload:
Once you've added photos, provide:
1. **School name:** INX Academy (already noted)
2. **Location:** City/Country where INX Academy is located
3. **Student feedback quotes:** 2-3 short testimonials (1-2 sentences each) with student first names if available
4. **Context:** Brief description of what's happening in each photo (optional)

## Example Feedback Format:
```
"BookBridge helped me improve my English reading so much. The audio feature is my favorite!"
- Sarah, 14, INX Academy

"I can finally understand difficult books. The instant translations save me so much time."
- Ahmed, 16, INX Academy
```
